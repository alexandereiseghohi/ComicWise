# Product Context

ProblemStatement: Describe the user problems ComicWise solves.

Users:

- Reader
- Creator
- Admin

UserGoals:

- Browse and read comics easily
- Creators upload/manage content
- Admins moderate and manage data

HighLevelFlows:

1. User signs in -> browse comics -> read chapter -> bookmark
2. Creator uploads comic -> images saved to public/comics -> metadata inserted
   via seeder

Constraints:

- Next.js 16 App Router
- Performance & SEO
